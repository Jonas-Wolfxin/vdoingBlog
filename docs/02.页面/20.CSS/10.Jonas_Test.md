## —— 纪伯伦
### 第1一次，当它本可进取时，却故作谦卑；
### 第二次，当它在空虚时，用情欲来填充；
### 第三次，在困难和容易之间，它选择了容易；
### 第四次，当它犯了错，却借由别人也会犯错来宽慰自己；
### 第五次，它萎靡软弱，却将其视之为生命的坚韧；
### 第六次，当它鄙夷一张丑恶的嘴脸时，却不知那正是自己面具中的一副；
### 第七次，它苟身于生活的污泥中，虽不甘心，却又畏首畏尾。
